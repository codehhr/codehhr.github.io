css常用居中方式

发表于2020-07-27|更新于2021-07-20|css
|字数总计:740|阅读时长:3分钟|阅读量:

一、水平居中

内联元素

父级元素加 text-align: center 即可

块级元素

给定宽度,然后 margin 上下为 0,左右 auto 即可

多个块级元素

第一种方式

子元素设置成内联,父级元素加 text-align:center即可

第二种方式

利用 flexbox 弹性布局

二、垂直居中

内联元素

第一种方式

设置 padding

第二种方式

按照父级元素的高度,设置子元素的行高

第三种方式

利用 flexbox,父级元素需给定高度

块级元素

第一种方式

父元素相对定位 position:relative,子元素绝对定位 position: absolute

第二种方式

利用 flexbox

三、水平和垂直居中

第一种方式

父元素相对定位 position:relative,子元素绝对定位 position: absolute

第二种方式

使用 flexbox

最后说一点,如果具体宽高已知,给定具体数值也可以直接实现
